# Gewenbo Blog

A bilingual blog on AI, technology, and culture.

[![Live Demo](https://img.shields.io/badge/demo-gewenbo--blog.vercel.app-111)](https://gewenbo-blog.vercel.app)

## Overview

A bilingual (English / Chinese) blog covering AI tools, emerging technology, cross-cultural comparisons, and the intersection of Eastern philosophy and modern science.

## Design & Tech

- **Stack:** Single HTML file, vanilla JavaScript
- **Deployment:** Vercel
